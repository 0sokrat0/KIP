## Описание структуры:

### 1. Начало
- Точка входа в программу

### 2. Объявление переменной
- `int choice` - переменная для хранения выбора пользователя

### 3. PrintMenu
- Вывод главного меню на экран:
  - 1. Добавить товар
  - 2. Удалить товар
  - 3. Заказать поставку товара
  - 4. Продать товар
  - 5. Поиск товаров
  - 6. Показать все товары
  - 7. Отчёт о продажах
  - 8. Отменить последнюю продажу
  - 9. История продаж
  - 0. Выход

### 4. Ввод choice
- Получение выбора пользователя с клавиатуры

### 5. switch(choice)
- Проверка значения choice и переход к соответствующей операции

### 6. Операции:
- **case 1**: Добавить товар
- **case 2**: Удалить товар
- **case 3**: Заказать поставку товара
- **case 4**: Продать товар
- **case 5**: Поиск товаров
- **case 6-9**: Другие операции (показать все, отчёт, отмена, история)
- **case 0**: Сообщение о выходе → Конец
- **default**: Сообщение об ошибке

### 7. Проверка условия
- После выполнения операции проверяется `choice == 0`
- Если `True` → Конец
- Если `False` → Возврат к PrintMenu (цикл продолжается)

## Особенности реализации:

- Все операции (кроме выхода) возвращаются к проверке условия
- Только при выборе "0" программа завершается
- При неверном выборе выводится сообщение об ошибке и цикл продолжается
- Программа работает в цикле до явного выхода пользователя
