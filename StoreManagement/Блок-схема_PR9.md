# Блок-схема практической работы №9
## Система управления магазином

```mermaid
flowchart TD
    Start([Начало]) --> Declare[int choice]
    Declare --> Init[Инициализация Store<br/>Загрузка тестовых данных]
    Init --> Menu[PrintMenu<br/>Вывод меню на экран]
    Menu --> Input[Ввод choice]
    Input --> Switch{switch choice}
    
    Switch -->|1| Add[Добавить товар]
    Switch -->|2| Delete[Удалить товар]
    Switch -->|3| Order[Заказать поставку товара]
    Switch -->|4| Sell[Продать товар]
    Switch -->|5| Search[Поиск товаров]
    Switch -->|6| ShowAll[Показать все товары]
    Switch -->|7| Report[Отчёт о продажах]
    Switch -->|8| Undo[Отменить последнюю продажу]
    Switch -->|9| History[История продаж]
    Switch -->|0| ExitMsg[Сообщение о выходе]
    Switch -->|default| Error[Сообщение об ошибке]
    
    Add --> Check
    Delete --> Check
    Order --> Check
    Sell --> Check
    Search --> Check
    ShowAll --> Check
    Report --> Check
    Undo --> Check
    History --> Check
    Error --> Check
    
    Check{choice == 0?}
    Check -->|True| End([Конец])
    Check -->|False| Wait[Ожидание нажатия клавиши]
    Wait --> Clear[Очистка экрана]
    Clear --> Menu
    
    ExitMsg --> End
```

## Описание блок-схемы

### Основные блоки:

1. **Начало** - точка входа в программу
2. **Объявление переменной** - `int choice` для хранения выбора пользователя
3. **Инициализация** - создание объекта Store и загрузка тестовых данных
4. **PrintMenu** - вывод главного меню на экран
5. **Ввод choice** - получение выбора пользователя
6. **switch (choice)** - проверка выбора пользователя

### Операции меню:

- **1** - Добавить товар
- **2** - Удалить товар  
- **3** - Заказать поставку товара
- **4** - Продать товар
- **5** - Поиск товаров
- **6** - Показать все товары
- **7** - Отчёт о продажах
- **8** - Отменить последнюю продажу
- **9** - История продаж
- **0** - Выход
- **default** - Сообщение об ошибке

### Цикл работы:

После выполнения операции (кроме выхода):
1. Проверка условия `choice == 0`
2. Если `False` - ожидание нажатия клавиши, очистка экрана, возврат к меню
3. Если `True` - завершение программы
